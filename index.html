PROPOSITO DE LAS ESTRUCTURAS LOGICAS
Las estructuras lógicas son aquellas que nos permiten controlar el acceso a determinadas partes del algoritmo.

Los condicionales son estructuras lógicas que permiten tomar decisiones y delimitar las operaciones que se pueden realizar.

Case es una estructura lógica que se utiliza para trabajar con elecciones únicas, es especialmente útil para realizar menús o controles de movimiento.

ELEMENTOS DE UN CONDICIONAL
Afirmación: Es la que permite que un condicional haga su trabajo puesto que ella presenta un valor de verdad ya sea falso o verdadero.

Operador lógico: Es aquel que nos permite comparar cantidades, dicha comparación puede ser falsa o verdadera.

Conector lógico: Es aquel elemento que nos permite unir afirmaciones, los conectores son: Y, O.

Las estructuras lógicas se pueden utilizar para predecir escenarios y posibilidades, que nos pueden ayudar a tomar diferentes decisiones.

La vida está llena de condiciones, para realizar alguna cosa se debe cumplir una condición.
Por ejemplo: Una persona quiere comprar un celular, este no se le puede entregar sino da el excedente para los audifonos y el cargador.
